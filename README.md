# arduino-in-rust-led
===================

Я зробив проект мигаючий світлодіод на Arduino Mega 2560 в Rust

## Інструкції зі складання
1. Встановіть необхідні компоненти, як описано в [`avr-hal` README] (`avr-gcc`, `avr-libc`, `avrdude`, [`ravedude`]).

2. Запустіть `cargo build` для складання прошивки.

3. Запустіть `cargo run` для прошивки підключеної плати. Якщо `ravedude`
не може виявити вашу плату, перевірте її документацію за адресою <https://crates.io/crates/ravedude>.

4. `ravedude` відкриє сеанс консолі після прошивки, де ви зможете взаємодіяти з консоллю UART вашої плати.

[`avr-hal` README]: https://github.com/Rahix/avr-hal#readme
[`ravedude`]: https://crates.io/crates/ravedude

## Про світлодіод.
Світлодіод (LED — light-emitting diode) — це напівпровідниковий прилад, який перетворює електричний струм безпосередньо на світлове випромінювання (електролюмінесценція). Це ефективні, довговічні джерела світла, що майже не нагріваються, використовуються для освітлення та індикації. Світлодіоди випромінюють світло вузького спектра, колір якого залежить від хімічного складу матеріалу.
Основні характеристики світлодіодів:
-Принцип дії: світяться, коли струм проходить через -перехід (від анода до катода).
-Типи: індикаторні (низька потужність) та освітлювальні (висока яскравість).
-Переваги: висока енергоефективність, довговічність (до 10 років), миттєве вмикання, відсутність ртуті.
-Застосування: побутове освітлення, автомобільні фари, індикатори на приладах, екрани, вуличне освітлення.

## Про Arduino.
Arduino — це відкрита апаратно-програмна платформа для створення електронних пристроїв, робототехніки та систем «розумного дому». Вона складається з компактної плати мікроконтролера (наприклад, Arduino Uno) та середовища розробки Arduino IDE, що дозволяє програмувати пристрій мовою C/C++.
Основні характеристики:
-Простота: Ідеально підходить для початківців, любителів та професіоналів.
-Доступність: Має низьку вартість, відкритий вихідний код (open-source) та велику спільноту.
-Функціональність: Плати оснащені цифровими та аналоговими входами/виходами для підключення датчиків, двигунів та інших модулів.
-Застосування: Від автоматизації процесів та створення метеостанцій до керування роботами.

## Про Arduino Mega 2560.
Arduino Mega 2560 — це потужна плата мікроконтролера на базі ATmega2560, яка є розширеною версією Arduino Uno. Вона має 54 цифрові входи/виходи (14 ШІМ), 16 аналогових входів, 256 КБ флеш-пам'яті та 16 МГц процесор, що робить її ідеальною для складних проектів, 3D-принтерів та робототехніки.
Ключові особливості Arduino Mega 2560:
-Мікроконтролер: ATmega2560 (8-бітний).
-Пам'ять: 256 КБ флеш-пам'яті, 8 КБ SRAM, 4 КБ EEPROM.
-Цифрові контакти: 54 (з яких 14 можна використовувати як ШИМ-виходи).
-Аналогові входи: 16 (10-бітні).
-Інтерфейси: 4 апаратних UART (послідовні порти), I2C (SDA/SCL), SPI.
-Живлення: Рекомендовано 7–12 В, робоча напруга 5 В.
-Програмування: Здійснюється через Arduino IDE.

## Про Rust
Rust — це сучасна мова системного програмування, що фокусується на безпеці пам'яті, швидкодії та конкурентності без використання збирача сміття, часто виступаючи надійною заміною C/C++. На Rust можна програмувати мікроконтролери, включаючи Arduino, особливо плати на базі 32-бітних архітектур (ARM Cortex-M), хоча підтримка класичних 8-бітних AVR також існує, але вона складніша.
Основні моменти використання Rust з Arduino:
-Сумісність: Rust добре підходить для вбудованих систем (embedded) завдяки низькорівневому доступу до пам'яті та апаратного забезпечення.
-Платформи: Найкраще Rust працює на потужніших платах типу ESP32 або STM32, але можлива прошивка і Arduino Uno/Nano (AVR).
-Екосистема: Існують спеціальні інструменти, такі як avr-rust (для старіших Arduino) або cortex-m (для новіших), які дозволяють завантажувати Rust-код у мікроконтролери.
-Переваги: Ви отримуєте безпеку, яку гарантує компілятор Rust (відсутність помилок доступу до пам'яті), при швидкості, близькій до C++.
